# ManOxCo Platform Baseline Repository

This repository is the baseline starting point for the **ManOxCo Oxygen Reliability Transformation Program - PI-1**.

Its purpose is to help consulting teams understand the case, the program objective, and the expected delivery scope before building their own implementation.

## Repository Objective

The goal of this repository is to provide the minimum shared foundation for teams working on the ManOxCo PI-1 initiative, which focuses on preventing liquid oxygen (LOX) dry-out incidents in hospitals through data engineering, analytics, operational planning, and AI-assisted decision support.

This baseline is intended to:

- Explain the business problem and program context
- Provide sample data assets for the case study
- Define the architectural and governance direction for PI-1
- Serve as a reference point for backlog-driven delivery work

## What This Repository Contains

The repository currently includes:

- `data/`: sample datasets used for the case study
- `manoxco_use_case.md`: the case study in Spanish
- `manoxco_use_case_en.md`: the case study in English
- `manoxco_pi_1.md`: the detailed PI-1 baseline program document
- `LICENSE`: repository license information

## How Teams Should Use This Repository

Teams should use this repository as a baseline reference, not as a finished solution.

Each team is expected to continue the implementation work independently, using the program guidance, backlog direction, and documentation provided here. Teams may work:

- Within the MAHG Enterprise Data Lab organization
- Locally in their own environment
- In their own GitHub accounts or repositories

## Project Template

A project template is available in GitHub Projects at:

<https://github.com/orgs/MAHG-Enterprise-Data-Lab/projects/4>

Teams may copy this project template to accelerate their setup. Some project entries have already been created, but each team is responsible for continuing the work, refining the backlog, and completing delivery based on its own execution approach.

Teams may continue this work either within the organization or locally in their own GitHub accounts.

## Reference Document

For the full PI-1 program baseline, architecture direction, governance expectations, and workstreams, see:

- `manoxco_pi_1.md`

## Program Outcome

The expected outcome is an end-to-end proposal or implementation that helps ManOxCo improve oxygen supply reliability, reduce operational risk, and support better planning and decision-making across production, logistics, hospital consumption, and financial analysis.
